# stream-live-to-instagram
stream any video live to Instagram

# Usage
 run :
 ```
 npm i
 ```
 to install, then enter your instagram account `username` and `password` in `config.js` file. run:
 ```
 npm start
 ```
 enter video file path when asked and __done !__
